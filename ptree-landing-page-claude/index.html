See it in action

terminal
$ ptree gen ./my-project --style unicode --profile spec

# Generates a .ptree file from your filesystem
# Respects .gitignore automatically
# Outputs to stdout or --output file

✓ Generated: my-project.ptree (42 entries)
terminal
$ ptree validate docs/structure.ptree

✗ PT003: Directory missing trailing slash at line 12
✗ PT007: Name type mismatch at line 15
⚠ PT009: Entries not sorted alphabetically

Found 2 errors, 1 warning
terminal
$ ptree validate docs/structure.ptree --fix --write

✓ Fixed: PT003 - Added trailing slash to "Components"
✓ Fixed: PT009 - Sorted 3 entries alphabetically

Applied 2 fixes. 1 issue requires manual review.
